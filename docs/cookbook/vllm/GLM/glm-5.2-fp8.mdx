# Deploying GLM-5.2-FP8 with GPUStack and vLLM

This tutorial guides you through setting up **GLM-5.2-FP8**—the official unquantized/FP8 Mixture-of-Experts model with massive long-context capabilities—using **vLLM (v0.24.0)** in the **GPUStack** Custom Container mode. 

---

### 1. Base Image
* **Image Name**: `swr.cn-south-1.myhuaweicloud.com/gpustack/vllm-openai:v0.24.0-cu129`
* **Hardware Requirements**: H20

### 2. Model Weights Source
* **Model Name**: `GLM-5.2-FP8`
* **Download Source (ModelScope)**: `https://modelscope.cn/models/ZhipuAI/GLM-5.2-FP8`

---

## 🚀 GPUStack Configuration Steps

Navigate to your GPUStack admin dashboard, go to Models -> deployment, and select the model deploy / modelscope mode.

### 1. Basic Information
* **Model Name**: `GLM-5.2-FP8`
* **Container Image**: `swr.cn-south-1.myhuaweicloud.com/gpustack/vllm-openai:v0.24.0-cu129`

### 2. Backend Arguments 
You can directly copy and paste the following parameters into the **server Arguments** text area in GPUStack (each argument occupies a single line without backslashes):

```text
--tensor-parallel-size 8
--speculative-config.method mtp
--speculative-config.num_speculative_tokens 5
--gpu-memory-utilization 0.8
--reasoning-parser glm45
--tool-call-parser glm47
----max-model-len 800000
--kv-cache-dtype fp8
--cuda-graph-max-bs 32
--enable-expert-parallel
--max-num-seqs 30
```
