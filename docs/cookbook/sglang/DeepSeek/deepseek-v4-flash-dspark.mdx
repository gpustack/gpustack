## 🚀 GPUStack Configuration Steps

Navigate to your GPUStack admin dashboard, go to **Models** -> **deployment**, and select the **model deploy** / **modelscope** mode.

### 1. Basic Information
* **Model Name**: `DeepSeek-V4-Pro-DSpark`
* **Container Image**: `swr.cn-north-4.myhuaweicloud.com/desaysv/gpustack/sglang-dspark:v1.0`

### 2. Backend Arguments (Multi-line Format for UI Input)
You can directly copy and paste the following parameters into the **server Arguments** text area in GPUStack (each argument occupies a single line without backslashes):

```text
--trust-remote-code
--tp 8
--ep-size 8
--moe-runner-backend flashinfer_mxfp4
--speculative-moe-runner-backend flashinfer_mxfp4
--speculative-algorithm DSPARK
--speculative-eagle-topk 1
--speculative-num-steps 1
--mem-fraction-static 0.85
--context-length 1000000
--cuda-graph-max-bs 32
--max-running-requests 32
--disable-overlap-schedule
```
